# `Tucanos`

![Tucanos](src/images/tucano.svg)

Tucanos is a 2D and 3D metric-based adaptation library based on [*Four-Dimensional Anisotropic Mesh Adaptation for
Spacetime Numerical Simulations* by
Philip Claude Caplan](https://www.cs.middlebury.edu/~pcaplan/docs/Caplan_2019_PhD.pdf).

The public repository containing the sources can be found [here](https://github.com/tucanos/tucanos).

This document briefly presents the main theoretical aspects and functionnalities that are offered by this remesher.
